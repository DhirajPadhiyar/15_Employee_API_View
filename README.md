# Employee Management System (MVC View + API)

A simple Employee Management System built using ASP.NET Core MVC, Entity Framework Core, SQL Server, and REST API architecture.

## Features

* Employee Create
* Employee List
* Employee API (GET)
* Employee API (POST)
* Service Layer Architecture
* Entity Framework Core
* SQL Server Integration
* Swagger API Documentation

## Technologies Used

* ASP.NET Core MVC
* C#
* Entity Framework Core
* SQL Server
* Swagger
* Bootstrap 5

## Project Structure

MVC Controller → Service Layer → Entity Framework Core → SQL Server

API Controller → Service Layer → Entity Framework Core → SQL Server

## API Endpoints

### Get All Employees

GET

```text
/api/EmployeesApi
```

### Create Employee

POST

```text
/api/EmployeesApi
```

## Getting Started

1. Clone the repository
2. Update the connection string in appsettings.json
3. Run migrations
4. Run the application
